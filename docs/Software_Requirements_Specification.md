INTRODUCTION
The Hospital Appointment API provides a secure REST API that enables hospitals and clinics to manage patients, doctors, and appointments efficiently.
Objectives
-Reduce manual appointment booking.
-Prevent double booking.
-Secure patient information.
-Provide role-based access.
-Demonstrate backend engineering skills.

2. SCOPE
   For Version 1:
-Authentication
-Doctor Management
-Patient Management
-Appointment Booking
-Search
-Pagination
-Filtering
